# Final Project
A simple web application that analyzes text and detects emotions using a natural language processing API.

The application allows users to input text and returns scores for the following emotions:
- Anger
- Disgust
- Fear
- Joy
- Sadness

It also identifies the **dominant emotion** based on the highest score.

---

## 🧱 Structure

- `emotion/` – core logic for calling the API and processing results  
- `server.py` – Flask application handling requests and responses  
- `templates/` – HTML frontend  
- `static/` – JavaScript for client-side interaction  
- `tests/` – unit tests with mocked API responses
